#  Clothing-Xpress ‚Äì Premium Hoodie Store

**clothing-Xpress** is a high-performance, modern e-commerce platform dedicated to premium streetwear. Built with a focus on speed and user experience, it allows users to browse, search, and manage a shopping cart with seamless transitions.

[ **View Live Demo**](https://clothing-xpress-9q3q.vercel.app/)

---

##  Tech Stack

| Technology | Use Case |
| :--- | :--- |
| **Next.js 14** | App Router, SSR, and optimized routing |
| **React 18** | Functional components and Hooks |
| **Tailwind CSS** | Responsive utility-first styling |
| **Context API** | Global state management for the Cart |
| **Lucide React** | Clean, lightweight iconography |

---

##  Key Features

*  **Real-time Search:** Instantly filter the hoodie collection as you type.
*  **Advanced Cart System:** Add, remove, and manage items with persistent state.
*  **Dynamic Collections:** Organized categories for Men, Women, and Unisex styles.
*  **Mobile-First Design:** Fully responsive layout optimized for all screen sizes.
*  **Hero Slider:** Interactive auto-sliding background showcasing featured arrivals.
*  **Sleek UI:** Modern aesthetics with a focus on high-quality product imagery.
*  **Interactive Forms:** Functional contact section with loading states and toast notifications.

##  Getting Started

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/clothing-xpress.git](https://github.com/your-username/clothing-xpress.git)
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set up environment variables:**
    Create a `.env.local` file for any API keys 
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
5.  **Build for production:**
    ```bash
    npm run build
    ```
